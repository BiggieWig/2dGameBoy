# Blue Boy Adventure

**Blue Boy Adventure** is a 2D top-down Role Playing Game (RPG) developed in Java. The game features exploration, real-time combat, puzzle-solving, and a complete inventory system.

## üéÆ Features

* **Open World Exploration:** Explore diverse areas including an overworld, interiors, and dungeons.
* **Real-Time Combat:** Fight slimes, orcs, and bosses using swords, axes, and fireballs.
* **Inventory System:** Manage weapons, shields, and consumables (potions, keys) via a dedicated character screen.
* **RPG Elements:** Level up system, stat management (Strength, Dexterity, Magic), and equipment slots.
* **Interactive Environment:** Cut down trees, break walls, and solve puzzles.
* **Dynamic Lighting:** Includes a day/night cycle and lantern mechanics for dark areas.
* **Save & Load:** Persist your progress (stats, inventory, and location) between sessions.
